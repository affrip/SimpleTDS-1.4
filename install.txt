Установка Simple TDS v1.3beta:

Скопировать все файлы на сервер.
Папке archive, и ВСЕМУ что внутри нее присвоить права 777

Создать на хостинге базу данных MySQL и залить в нее дамп dump.sql. Все эти манипуляции очень просто проделывать с помощью утилиты phpMyAdmin.

В файле config.php прописать следующие данные: пароль для доступа к админке скрипта, данные для соединения с базой MySQL, урл скрипта (как он будет виден из веб), резервный урл, на который будет литься траф при ошибках, и определить действия TDS при отсутствии GeoIP на сервере.

При начальной настройке можно выставить $debug = "true"; - при этом при обращении к go.php редиректа не будет, а будет выдаваться отладочная информация. В дальнейшем же, при работе с траффом, $debug должен быть выставлен в "false"!!!

Все, можно зайти через браузер по урлу скрипта, ввести указанный пароль и пользоваться скриптом.
